# Discord Bot written in C# with fully functional Application Commands

## Requirements
- Docker

## Installation steps
- Make a copy of `docker-compose.yml.example` called `docker-compose.yml`
  - Set the fields as desired
- Run `docker compose up -d` to launch the bot

## Optional
- To run only in certain servers you can add `TEST_SERVERS: "your guild id"`
- More guild ids can be added like so `TEST_SERVERS: "123456789, 987654321"`
